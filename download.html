Me and My Shadow can be downloaded for multiple platforms.

Windows (0.5a, .7z)

Extract archive and launch meandmyshadow.exe

Mac OS X (0.4, .dmg)

Open .dmg file and drag-and-drop Me And My Shadow to Applications folder. Launch the game from Applications folder.

Linux binary (0.5a, .AppImage)

chmod a+x and execute. You may need to install the package libwayland-egl1-mesa.

Check if there's a package available for your distribution.

Source package (0.5a, .tar.gz)

You can also download at Github.

Read our building guide

Older releases can be found here: https://sourceforge.net/projects/meandmyshadow/files/

Nightly Builds

To play with the latest developing cool features, you can build the game from source or download prebuilt nightly builds.

Windows binary (nightly, .7z)

Please download the bin+data version. (Get the data folder from GitHub master branch if you downloaded binonly version.) Extract archive and launch meandmyshadow.exe. You may need the Visual C++ 2013 redistributable.

You can find previous versions of Windows nightly build on AppVeyor. Click Configuration: Release then ARTIFACTS to download the nightly build.

Linux binary (nightly, .AppImage)

chmod a+x and execute. You may need to install the package libwayland-egl1-mesa.

GitHub repository

Or source package from GitHub master branch.
